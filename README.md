# 📝 x-writer

## 🦄 Features

- Next.js 13
- React 18
- TypeScript 4
- Tailwind CSS 3
- Path Mapping (for absolute imports)
- OpenAI GPT-3

## 🚀 Quick start

To run this project locally:

```shell
npm i
npm run dev
```

## 🎓 Learning Next.js

Looking for more guidance? Full documentation available on the official [Next.js website](https://nextjs.org).
